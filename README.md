# CloudOps Dashboard

A Node.js and Express demo dashboard that simulates cloud operations monitoring.

## What it shows

- Server health
- Application status
- SSL certificate status
- Cloudflare status
- Security alerts
- Backup status

## Technologies

- Node.js
- Express
- Bootstrap
- GitHub

## Why I built it

I built this project to strengthen my hands-on technical understanding of cloud infrastructure, application monitoring, and customer-facing technical enablement.

## Run locally

npm install
node app.js

Open http://localhost:3000
